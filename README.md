# Julian Tiqui — Personal Portfolio

A modern personal portfolio website built with **Next.js**, designed to showcase my resume, projects, experience, and technical work in one clean, fast, and professional space.

The visual direction is inspired by a mix of **Apple-style Liquid Glass UI**, dark automotive styling, and the aggressive modern look of the **2025 S650 Mustang GT Black**. Think: clean interface, glossy depth, sharp contrast, and just enough drama without becoming a Fast & Furious loading screen.

## Overview

This site is being built as my central online presence. It will eventually include:

* A polished resume section
* Featured engineering and software projects
* Links to GitHub, LinkedIn, and other professional profiles
* A personal/about section
* Visual elements inspired by my Mustang GT
* Possibly investing-related content or Blossom profile links
* A clean deployment pipeline through Vercel

## Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** CSS / Tailwind CSS
* **Deployment:** Vercel
* **Version Control:** Git + GitHub

## Current Goals

The first version of the site focuses on getting the foundation right:

1. Build a strong landing page
2. Add resume content in a clean, readable format
3. Create a projects section
4. Add contact/social links
5. Refine the design with a dark, glassy, modern aesthetic
6. Deploy publicly through Vercel

## Design Direction

The site should feel:

* Modern
* Sharp
* Professional
* Fast
* Slightly futuristic
* Clean, but not boring
* Automotive-inspired without looking like a car dealership website

Main visual ideas:

* Dark backgrounds
* Glassmorphism / Liquid Glass-style panels
* Subtle gradients
* Metallic accents
* Smooth motion
* Strong typography
* High contrast sections
* Optional hero image featuring my car

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Deployment

This project is intended to be deployed with **Vercel**.

Typical deployment flow:

```bash
git add .
git commit -m "Update portfolio"
git push
```

Vercel will automatically redeploy the site when changes are pushed to the main branch.
